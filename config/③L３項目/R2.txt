! ============================================================
! ③ L３項目 : R2 （拠点間接続ルータ）
! 出典: 設計書_md/03_論理設計.md（論理設計⑥⑦）, 08_拠点間接続設計.md
! 内容: CSW接続L3 / IP-VPN側IP / ループバック
! 前提: OSPF は ④、BGP は ⑦ で設定
! ============================================================
!
! ------------------------------------------------------------
! ■ 不明点・確認事項（実機設定時に確認）
!  - Gi0/0 → CSW2 FE0/6（10.1.255.24/30, R2=.25）。物理設計⑦: CSW接続=Gi0/0。
!  - Gi0/1 → IP-VPN網（192.168.12.2/30）。対向キャリア側=192.168.12.1。
!  - ルータのインターフェースは既定shutdownのため no shutdown を明示。
!  - ループバック 10.1.254.6/32 は OSPF router-id / iBGPピア源に使用（④⑦）。
! ------------------------------------------------------------

hostname R2
!
! ============================================================
! ループバック（論理設計⑦: R2 = 10.1.254.6/32）
! ============================================================
interface Loopback0
 ip address 10.1.254.6 255.255.255.255
!
! ============================================================
! CSW2 接続（L3 / 論理設計⑥: 10.1.255.24/30 R2=.25）
! ============================================================
interface GigabitEthernet0/0
 ip address 10.1.255.25 255.255.255.252
 no shutdown
 description To CSW2 FE0/6
!
! ============================================================
! IP-VPN網 接続（論理設計⑦ / 拠点間接続設計②: 192.168.12.2/30）
! ============================================================
interface GigabitEthernet0/1
 ip address 192.168.12.2 255.255.255.252
 no shutdown
 description To IP-VPN (peer 192.168.12.1)
!
end
