⑦ EGPルーティング : 確認コマンド（show系）
出典: 設計書_md/08_拠点間接続設計.md / 設問 17-22
対象機器: R1, R2, CSW1, CSW2, DSW1, DSW2

【共通の確認観点】
- eBGP(R1/R2 ↔ IP-VPN)がEstablishedか（認証cisco123一致）
- iBGPフルメッシュ（6ノード=15セッション）が全てEstablishedか
- 集約10.1.0.0/16がIP-VPNへ広告され、詳細(10.1.x.0/24)が漏れていないか
- DSW/CSWのBGP表に192.168のルートが「無い」か（設問19条件）
- next-hop-selfにより拠点間経路(10.2/10.3)がCSW/DSWで有効(>)になっているか
- 経路調整: 10.2→R2優先, 10.3→R1優先(Local Pref200), 戻りR1優先(prepend)

【iBGPフルメッシュ = 完全グラフ(6ノード)】
  ノード: DSW1(.1) DSW2(.2) CSW1(.3) CSW2(.4) R1(.5) R2(.6)  ※10.1.254.x
  各ノードが他5ノードとピア → 15本のiBGPセッション


------------------------------------------------------------------
■ R1 / R2（eBGP + iBGP + 集約 + 経路調整）
------------------------------------------------------------------
show ip bgp summary                 ! eBGP(192.168.x.1)とiBGP5本がEstablishedか
show ip bgp neighbors 192.168.11.1  ! eBGP状態・認証・route-map(in/out) ※R1の例
show ip bgp                          ! BGP表全体（10.1.0.0/16, 10.2.0.0/16, 10.3.0.0/16）
show ip bgp 10.1.0.0/16              ! 自拠点集約がlocalorigin(network)で広告されるか
show ip route 10.1.0.0              ! Null0への集約静的（方式1の前提）
show ip bgp 10.2.0.0                ! R2でLocalPref200/R1では100（拠点2）
show ip bgp 10.3.0.0                ! R1でLocalPref200/R2では100（拠点3）
show route-map                       ! SET-LOCALPREF-IN / PREPEND-OUT(R2のみ)

  ▼ 期待値
   - eBGP 192.168.x.1 が Established、iBGP 5ピアも Established
   - R1: 10.3.0.0/16 の Local Pref=200、R2: 10.2.0.0/16 の Local Pref=200
   - R2からIP-VPNへの10.1.0.0/16広告のAS-PATHが自AS3回分長い（show ip bgp
     neighbors 192.168.12.1 advertised-routes で確認）
   - 192.168.x.0/30 はBGP表に載っていない（network未指定のため）


------------------------------------------------------------------
■ CSW1 / CSW2 / DSW1 / DSW2（iBGPフルメッシュ）
------------------------------------------------------------------
show ip bgp summary                 ! 他5ノードとのiBGPが全てEstablishedか
show ip bgp                          ! 10.2.0.0/16, 10.3.0.0/16 を受信しているか
show ip bgp 10.2.0.0                ! best pathがR2(10.1.254.6)方向, LocalPref200
show ip bgp 10.3.0.0                ! best pathがR1(10.1.254.5)方向, LocalPref200
show ip route bgp                    ! 拠点間経路がRIBにインストールされ有効(>)か
show ip bgp | include 192.168        ! 何も出ないこと（192.168はBGP表に無い/設問19）

  ▼ 期待値
   - iBGP 5ピア全て Established
   - 10.2.0.0/16 の next-hop=R2 Lo0(10.1.254.6)、10.3.0.0/16 の next-hop=R1 Lo0
   - next-hop がOSPFで解決でき、経路が有効（next-hop-self が効いている証拠）
   - 192.168.x.x は表示されない
