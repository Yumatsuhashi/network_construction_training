! ============================================================
! ③ L３項目 : R1 （拠点間接続ルータ）
! 出典: 設計書_md/03_論理設計.md（論理設計⑥⑦）, 08_拠点間接続設計.md
! 内容: CSW接続L3 / IP-VPN側IP / ループバック
! 前提: OSPF は ④、BGP は ⑦ で設定
! ============================================================
!
! ------------------------------------------------------------
! ■ 不明点・確認事項（実機設定時に確認）
!  - Gi0/0 → CSW1 FE0/6（10.1.255.20/30, R1=.21）。物理設計⑦: CSW接続=Gi0/0。
!  - Gi0/1 → IP-VPN網（192.168.11.2/30）。対向キャリア側=192.168.11.1。
!  - ルータのインターフェースは既定shutdownのため no shutdown を明示。
!  - ループバック 10.1.254.5/32 は OSPF router-id / iBGPピア源に使用（④⑦）。
! ------------------------------------------------------------

hostname R1
!
! ============================================================
! ループバック（論理設計⑦: R1 = 10.1.254.5/32）
! ============================================================
interface Loopback0
 ip address 10.1.254.5 255.255.255.255
!
! ============================================================
! CSW1 接続（L3 / 論理設計⑥: 10.1.255.20/30 R1=.21）
! ============================================================
interface GigabitEthernet0/0
 ip address 10.1.255.21 255.255.255.252
 no shutdown
 description To CSW1 FE0/6
!
! ============================================================
! IP-VPN網 接続（論理設計⑦ / 拠点間接続設計①: 192.168.11.2/30）
! ============================================================
interface GigabitEthernet0/1
 ip address 192.168.11.2 255.255.255.252
 no shutdown
 description To IP-VPN (peer 192.168.11.1)
!
end
